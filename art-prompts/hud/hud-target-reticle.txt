HUD element: Target Reticle / Break Region Contrast-Ring. Canvas 128x128 px.
Flat UI overlay, transparent center.

This is the "contrast-ring pulse" described in art-bible §2.2 and §3.7:
"on cursor-rest/target-select, the part steps forward via a contrast-ring pulse —
a luminosity cue, greyscale-readable." The reticle lights up the seam/boundary of
the targeted break region when the player hovers over a target choice.

ANATOMY:
A chamfered rectangular ring (NOT a circular crosshair). The ring is sized to
overlay a break region (a sub-portion of the enemy sprite, roughly 1/3 of the bot).
Center is fully transparent — only the ring border is visible.

RING GEOMETRY:
- Chamfered rectangular ring: straight sides with 45-degree chamfered corners
  (~12px diagonal chamfer at each corner)
- Ring border width: 3-4px on each side
- INNER dimensions: approximately 80x80 px of transparent center within the 128x128 canvas
  (24px border on all sides, adjusted for chamfer geometry)
- The ring is RECTILINEAR with chamfered corners — following the §3.6 "hard edges = UI"
  rule. No circular reticle. The game targets regions as rectangular zones.

STATES (generate 2 variants):

STATE 1 — NORMAL TARGET (cursor resting on a break region):
- Ring color: C-6 UI White #E8E8E8 at ~70% opacity (high contrast against the
  desaturated combat background — luminosity cue, greyscale-readable)
- Ring has a soft outer glow: 2px blur at 30% opacity same color (pulse captured mid-cycle)
- Non-color channel: the ring shape itself IS the non-color channel

STATE 2 — SELECTED TARGET (player has committed to this target):
- Ring color: Info Blue #4090CC at ~90% opacity (selection color per §4.3)
- Slightly thicker ring: 4-5px border
- Slightly stronger outer glow at Info Blue

CRITICAL DISTINCTION: The ring must NOT use element colors (Volt cyan, Thermal amber,
Kinetic silver). Element identity is read from the element badge (§4.5), not from
the targeting reticle. The reticle uses only Chrome colors (white or Info Blue) so it
never creates a false element read.

GREYSCALE TEST: The ring border is fully readable in greyscale. The shape of the
chamfered rectangular ring communicates "this region is selected" without color.

NEGATIVE: NO circular crosshair reticle (this is a chamfered rectangular ring, not
a circle). NO element colors on the ring itself. No solid fill in the center (alpha
transparent). No organic curves. No photoreal. No painterly gradients.
