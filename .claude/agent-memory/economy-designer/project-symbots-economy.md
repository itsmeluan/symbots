Scrap is the sole currency (HOLISM-01 decision). MVP faucet ~1,840 Scrap over ~200 victories (band ~1,555–2,125). Sink = material-gated part upgrading; full 3-Symbot priority sink ~3,000. Funds ~1.8 loadouts → mild scarcity (2 Symbots nearly kitted, third partially).

**Part scrap yields**: Common 5 / Rare 20 / Prototype 35 / Boss-grade 60.

**Upgrade cost curve** (proposed, owned by Part Upgrade/Workshop): 0→1: 10, 1→2: 20, 2→3: 40, 3→4: 80, 4→5: 160 (pure doubling). Cap totals: Common +3 = 70; Rare/Prototype/Boss-grade +5 = 310.

**Base drop rates**: Common 0.70, Rare 0.25, Prototype 0.05, Boss-grade 0.001.

**Pity**: N_PROTO_PITY = 25 (Prototype gradient), M_BOSS_PITY = 8 (Boss-grade deterministic floor).

**MULTIPLIER_FLOOR = 1.5** — minimum drop-condition multiplier that counts as a real incentive.

**Why**: HOLISM-01 was the cross-review decision that resolved Scrap as the economy anchor. All economy numbers flowed from the 3-Symbot mild-scarcity brief.

**How to apply**: Any consumable buy/sell prices, drop faucets, or new currency sources must be evaluated against the ~1,840 Scrap arc faucet and the ~1,000/Symbot sink to confirm mild scarcity holds.
